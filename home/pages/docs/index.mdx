# Storyden documentation

Welcome to Storyden!

> Storyden is in an early testing and we're aiming to get as much feedback as possible so please feel free to share any thoughts, experiences, bugs or suggestions on [GitHub](https://github.com/Southclaws/storyden) or via this form: https://airtable.com/shrLY0jDp9CuXPB2X

## Get started

Storyden is the platform for managing your community and its content, wherever they call home. Run a forum, a blog, post news, aggregate links, build a database or a knowledgebase and more.

Deploy anywhere, [get started here](/docs/quickstart).
