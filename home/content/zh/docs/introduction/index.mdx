## 快速开始

准备好计时器，我们来快速启动一个社区。

Storyden 重视现代化和简单性。最快的体验方式是用 Docker 启动一个本地实例：

```sh
docker run -p 8000:8000 ghcr.io/southclaws/storyden
```

启动后，你可以在 `http://localhost:8000` 访问 Storyden 并进行体验。

![Storyden web UI](/docs/introduction/local/browser_home.png)

接下来，你可以注册一个新账户并探索平台。
