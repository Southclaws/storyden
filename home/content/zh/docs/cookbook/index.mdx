实战食谱收集了用 Storyden 搭建和运营社区的实用指南与模式。这些菜谱会展示如何把 Storyden 的功能组合起来，做出一些很顺手的工作流。

## 食谱里有什么？

和介绍、参考文档不同，食谱更关注如何把多个 Storyden 功能拼在一起使用。

每个食谱都尽量做到：

- **完整**：从头到尾展示整个流程
- **实用**：来自真实社区场景
- **可执行**：包含你可以直接照着做的具体步骤

## 贡献食谱

你的社区里有好用的模式或工作流吗？欢迎把它加进食谱！可以在 [GitHub](https://github.com/Southclaws/storyden) 上开 issue 或提交 pull request。
